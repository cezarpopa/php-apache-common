# PHP Apache Common

### A docker image with PHP, Apache and some common extensions and tools


Extensions:

* `pdo` 
* `pdo_mysql` 
* `mysqli`
* `zip` 
* `json` 
* `xml` 
* `mbstring`
* `gd`

composer: 
	
* `1.7.2`

tools:
	
* `nano` 
* `zip` 
* `unzip` 
* `git` 
* `iputils-ping` 
* `net-tools acl`


Pull from docker hub: pavlakis/php-apache-common:{PHP_VERSION}

e.g. pavlakis/php-apache-common:7.2.11
